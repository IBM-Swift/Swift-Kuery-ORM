  • idColumnName Default implementation

    Defines the id column name in the Database and if set to the same name as a field in the model it will make that field the Primary Key of the table in the Database.

    If that field is of type Int?, it will set it as auto increment in the Database and will populate the field when the model is returned.

    If when saving a model, that field has a value it will not save it but instead get an auto increment value.

    Default Implementation

    Defaults to id
